# Resume-Screening-App
Resume Screening App With Python and Machine Learning 

**Project Overview**
1. Take data present from Kaggle
2. Categories = Label. Explore
3. Data Cleaning > URLs, hashtags, special characters, punctuations
4. Vectorize using TFiDF first 200 words
5. KNeighbors classification, OneVsMany
6. Fit, predict, inverse label encoding
7. Build streamlit app
